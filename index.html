LambdaNeXt

Lambda NeXt is Lambda School's flagship program to help students build relevant projects to showcase their skills. The goal of Lambda NeXt is to build projects that solve real world problems by helping crowdsource volunteers for research.

  • Stack: React/Redux on Front-End, NodeJs/Express on Back-End and REST API with PostgreSQL as DMS.
  • Functioned as a Back-End Developer in a cross-functional team with 1 UX Designer, 2 Front-End and 3 Back-End Developers, to build the project in 8 weeks.
  • Collaborated with 2 Back-End Developers to build a REST API, assisted with front-end design efforts, and took lead on developing eye-catching logo.
  • Built REST API utilizing Express on NodeJs framework, set up authentication using PassportJs, and designed database schema with Postgres as DMS.

Dating Success

Dating Success rates users dating profiles, from 0 to 100, by comparing it to 60.000 OkCupid profiles following a set algorithm. The intention is to predict how successful the profile will be and guide users as they navigate the online dating world.

  • Stack: Python to create the algorithm, NodeJs/Express on Back-End with a REST API and SQL database, and ReactJs on Front-End.
  • Collaborated in a cross-functional team with 1 Data Scientist, 1 Back-End and 2 Front- End Developers, to build the project in 4 days.
  • Solely responsible for the back-end, setting up a REST API connected to the Python algorithm and designed database schema.
  • Built REST API utilizing Express on NodeJs framework to allow agile communication between the front-end and the Python algorithm.
